# Introduction

This is an application used to discover publicly available LLM routers over Syftbox.


# Installation

1. Run your syftbox client
2. Clone this repository
    ```bash
    git clone https://github.com/OpenMined/syft-llm-router.git
    ```
3. Install the `discover` app:
    ```bash
    syftbox client app install /path/to/syft-llm-router/discover
    ```
4. Visit the index.html page in your datasite's public folder under the syft_routers directory.
